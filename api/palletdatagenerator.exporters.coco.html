palletdatagenerator.exporters.coco module

COCO format exporter for object detection and segmentation annotations.

class palletdatagenerator.exporters.coco.COCOExporter(output_dir: str, dataset_name: str = 'PalletDataset')[source]

Bases: object

Export annotations in COCO (Common Objects in Context) format.

COCO format is widely used for object detection, segmentation, and keypoint detection. This exporter focuses on object detection with bounding boxes.
