Export object detection annotations in YOLO format.

YOLO format specifications: - One text file per image - Each line: class_id center_x center_y width height - All coordinates normalized to [0, 1]

class palletdatagenerator.exporters.COCOExporter(output_dir: str, dataset_name: str = 'PalletDataset')[source]

Bases: object

Export annotations in COCO (Common Objects in Context) format.

COCO format is widely used for object detection, segmentation, and keypoint detection. This exporter focuses on object detection with bounding boxes.
