palletdatagenerator.exporters.yolo module

YOLO format exporter for object detection annotations.

class palletdatagenerator.exporters.yolo.YOLOExporter(output_dir: str, class_mapping: dict[str, int] | None = None)[source]

Bases: object

Export object detection annotations in YOLO format.

YOLO format specifications: - One text file per image - Each line: class_id center_x center_y width height - All coordinates normalized to [0, 1]
